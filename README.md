
# zzsn

Awesome zzsn created by mwiszenko

## Install it from PyPI

```bash
pip install zzsn
```

## Usage

```py
from zzsn import BaseClass
from zzsn import base_function

BaseClass().base_method()
base_function()
```

```bash
$ python -m zzsn
#or
$ zzsn
```
